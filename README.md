# 🚀 AI Resume Analyzer with ATS Score

An **AI-powered Resume Analyzer** that evaluates resumes based on **ATS (Applicant Tracking System) standards**, analyzes content for specific job roles, and provides smart suggestions to improve selection chances.

---

## ✨ Features

- 📊 ATS Score Detection  
- 🎯 Target Job Role Selection  
- 📝 Resume Content Analysis (Paste Input)  
- 🤖 AI-Based Feedback & Suggestions  
- 🔍 Keyword Optimization for Job Roles  
- ⚡ Fast & User-Friendly Interface  

---

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** Node.js  
- **AI Integration:** OpenAI / Gemini API  

---

## ⚙️ How It Works

1. Select your **target job role**  
2. Paste your **resume content**  
3. Click on **Analyze Resume**  
4. Get:
   - 📊 ATS Score  
   - ❌ Missing Keywords  
   - ✅ Improvement Suggestions  
   - 📈 Resume Optimization Tips

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

cd ai-resume-analyzer

### 2️⃣ Install Dependencies

npm install

### 3️⃣ Setup Enviromental Variables

API_KEY=your_api_key_here

### 4️⃣ Run the Project

node server.js
